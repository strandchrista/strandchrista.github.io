## Hosting on GitHub Pages

1. Push all files to your GitHub repository (keep them all in the **same folder**).
2. Go to **Settings → Pages** in your repo.
3. Under *Source*, select your branch (e.g. `main`) and root folder `/`.
4. Save — your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Notes

- All styles and scripts are embedded in `index.html` — no build step needed.
- Google Fonts (DM Serif Display) are loaded via CDN and require an internet connection to render correctly.
